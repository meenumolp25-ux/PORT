PM Vikas Program

Pradhan Mantri Virasat Ka Samvardhan — a flagship initiative for the holistic development of minority communities through skill development, education, and livelihood support.

Empowering Communities Through Inclusive Development

The PM VIKAS program consolidates the key interventions of previously existing schemes, focusing on skill training, education, and economic empowerment of minority communities across India. It aims to provide holistic development through a multi-sectoral approach.

PM Vikas — Daily Class Journal (Day 1–17)

A concise learning journal covering electronics, networking, IoT, Arduino and more (May 20 — June 6, 2026).

Overview

This learning journal documents the Pradhan Mantri Vishwakarma Kaushal Samman (PM Vikas) programme at IIIT Kottayam covering fundamentals from electronics to Arduino and IoT.

Day 01: Resistors, Ohm's Law, KCL & KVL

Instructor: Dr. Anand — HOD ECE, IIIT Kottayam

Course Overview: Network Theory, Circuits & Networks, Basic Electronics, and Rectification. Real-world example: hardware inside a heart rate monitor.

Rectification & Charger Internals

Full-wave rectifier (bridge) producing pulsating DC, then filtered by RC / RLC / LC filters. RC filter often used to smooth output. Voltage dividers used to step voltages (e.g., 12V → 5V).

Resistors
  • No polarity; measured in Ohms.
  • Surface-mount resistors are common on modern PCBs.
  • Series: same current, R_total = R1 + R2 + ...
  • Parallel: same voltage, 1/R_total = 1/R1 + 1/R2 + ...
Ohm's Law & Kirchhoff

V = R × I. Ammeter in series; voltmeter in parallel. KCL: currents at a node sum to zero. KVL: voltage drops around a loop sum to zero.

Day 02: Diodes & Rectifiers

Instructor: Dr. Anand

Studied V–I characteristics of diodes, forward/reverse bias, diode as switch. Covered half-wave and full-wave rectifiers, clamper circuits and switch regulators.

Day 03: Computer Networks & VS Code Setup

Installed VS Code in a VM for development. Reviewed IEEE 802.3 (Ethernet) and IEEE 802.11 (Wi‑Fi). Covered the five components of data communication: Sender, Medium, Message, Receiver, Protocol.

Day 04: IoT Fundamentals & M2M Applications

Instructor: Shajulin Sir

Discussed M2M applications (healthcare, smart homes, IIoT, ITS, smart retail), IoT subsets (IIoT, CIoT, Space-IoT, IIoMT), and research challenges (trustworthy IoT with AI).

Day 05: IoT Product Ideation — Smart Home & Cybersecurity

Design task: smart home solution for elderly with integrated cybersecurity, SOS alerts, and smart access for delivery personnel.

Day 06: History of Electronics & Components

Instructor: Dr. Vineeth

Covered vacuum tubes → transistors → ICs timeline and differences between active (transistors, diodes) and passive (R, C, L) components. Introduced lab equipment: breadboard, multimeter, DSO, function generator, soldering kit.

Day 07: Digital Electronics — Logic Gates & Boolean Algebra

Analog vs Digital, logic gates (AND, OR, NOT, NAND, NOR, XOR, XNOR), number systems, Boolean laws, De Morgan's theorems, and Karnaugh maps for simplification.

Day 08: Combinational Circuits — Adders, MUX, Decoders

Half adder (S = A ⊕ B, C = A·B), full adder, subtractors, multiplexers/demultiplexers, encoders/decoders, and synthesis vs analysis concepts.

Day 09: Sequential Logic — Flip-Flops & State Machines

Synchronous vs asynchronous circuits, flip-flop types (SR, JK, D, T), design steps for state machines, state assignment strategies, and state reduction tips.

Day 10: Embedded Systems & Microcontrollers

Instructor: Dr. Della Thomas

Embedded system characteristics, microcontroller vs microprocessor, and ATmega328P specs (Arduino Uno).

Day 11: MCU Architecture, GPIO & Communication Protocols

Memory architectures (Von Neumann vs Harvard), GPIO basics, UART/SPI/I2C, interrupts, timers, ADC, and Arduino bridges.

Day 12: Networking & Cloud Basics — Part 1 (OSI & TCP/IP)

Walked through the OSI 7 layers, IP vs MAC addresses, ports, DNS, TCP vs UDP, and common network devices/topologies.

Day 13: Networking & Cloud Basics — Part 2 (IoT Protocols & Cloud)

IoT protocols: MQTT (pub/sub), HTTP/REST, CoAP, WebSocket. Cloud basics and IoT platform options (ThingSpeak, Blynk, Firebase). Service models: IaaS, PaaS, SaaS.

Day 14: Cisco Packet Tracer — Network Simulation Lab

Hands-on simulation: LANs, routers, subnetting, packet inspection, and commands like ping, ipconfig, and tracert.

Day 15: Introduction to AWS & Docker

Core IoT cloud services (EC2, S3, IoT Core, Lambda, DynamoDB) and Docker fundamentals (images, containers, Dockerfile).

Day 16: Arduino Programming — Setup, Pins & First Sketches

Arduino Uno anatomy, `setup()` and `loop()`, core I/O functions, and the first blink sketch:

void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}

Day 17: Sensors, Analog Input & Actuators

Analog reads (`analogRead(A0)` → 0–1023), PWM (`analogWrite`), common sensors (TMP36/LM35, HC-SR04, DHT11, LDR), and actuators (LEDs, buzzers, motors, servos).

Useful function: map() for scaling ADC values to servo angles.
